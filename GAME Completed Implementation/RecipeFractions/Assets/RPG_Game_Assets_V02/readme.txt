This 90+ piece Low Poly RPG Game Asset Pack (and growing) contains a wide variety of optimized game assets which has been textured, unwrapped, and UV mapped. The mesh and texture are very small and can be used in RPG games. The asset can be easily imported into Unity. You also get a Unity Package File, making it easier to jump directly into game development.

The new update also contains high resolution, quality textured models for 10+ models. - Enjoy!
